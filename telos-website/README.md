## 개요

주식회사 텔로스의 공식 회사 홈페이지입니다.

- **도메인**: https://www.telos.it.kr
- **호스팅**: GitHub Pages
- **DNS / SSL**: Cloudflare
- **이메일**: contact@telos.it.kr (Cloudflare Email Routing)

## 사업 영역

1. **AI 헬스케어** - 음성·행동·감정 데이터 기반 선제형 돌봄 솔루션
2. **AI SaaS 솔루션** - NLP, 음성 인식, 데이터 분석 클라우드 서비스
3. **정보통신 인프라** - 안정적 데이터 처리 + 보안 백엔드 시스템
4. **R&D 협력** - 산학연 공동 연구 및 기술 상용화

## 파일 구조

```
telos-website/
├── index.html          메인 페이지
├── css/
│   └── style.css       메인 스타일시트
├── js/
│   └── main.js         메인 JavaScript
├── images/             배경 이미지
├── CNAME               커스텀 도메인 연결
├── robots.txt          검색엔진 안내
└── README.md           본 파일
```

## 기술 스택

- HTML5 + CSS3 + Vanilla JavaScript (빌드 도구 없음, 정적 사이트)
- Pretendard Variable (한글 폰트, CDN 제공)
- 반응형 디자인 (모바일/태블릿/데스크톱)
